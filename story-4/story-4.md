## Installation d'Apktool
 
### 1. Téléchargement
Fichiers récupérés sur [apktool.org](https://apktool.org/docs/install) :
- `apktool.bat`
- `apktool.jar` (renommé exactement `apktool.jar`)

## Décompilation de l'APK
 
### Commande utilisée
```powershell
apktool d "chemindelapk\app.apk" -o "chemindudossier\story-4" -f
```

## Résultat 

```
Apk décompilé puis zippé manuellement
```
